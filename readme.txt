Simplicy twitter press is a widget for your sidebar that displays your latest tweets, follow me button, and users who follow you with their picture.

== Description ==

<strong>Simplicy twitter press</strong> The options:
 
<p>Size of your picture User<br />
Display your latest tweets: Yes / No<br />
Number of tweets posted<br />
Show users who follow you: Yes / No<br />
Max number of users has preloaded<br />
Size of user picture<br /><br />
<strong>Language:</strong> french</p>


== Installation ==

Extract the zip file and just drop the contents into the folder wp-content/plugins / directory of your WordPress installation and activate the plugin from the Plugins page.
